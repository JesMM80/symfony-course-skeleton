Hay que instalar NODE 14 and Yarn. Ponemos estas instrucciones en el dockerfile bajo la línea 29. 
RUN curl -sL https://deb.nodesource.com/setup_14.x | bash -
RUN apt -y install nodejs
RUN curl -sL https://dl.yarnpkg.com/debian/pubkey.gpg | apt-key add -
RUN echo "deb https://dl.yarnpkg.com/debian/ stable main" | tee /etc/apt/sources.list.d/yarn.list
RUN apt update && apt install yarn
Después ejecutamos make build, make restart. A continuación ejecutamos dentro del contenedor:

composer require symfony/polyfill-php80:^1.25 --no-update
composer update symfony/polyfill-php80 symfony/webpack-encore-bundle
composer require symfony/webpack-encore-bundle:^1.16
actualizado el package.json con las versiones aptas para el curso
